# tessle

My personal reimplementation of Wordle.
See: [https://tessle.tessmakes.at/](https://tessle.tessmakes.at/)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
